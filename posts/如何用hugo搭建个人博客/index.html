如何用hugo搭建个人博客

本地安装git

windows系统

一. 安装hugo

  1. 选择自己电脑对应的版本进行下载
  2. 下载安装完成之后配置好环境变量
  3. 到终端输入 hugo version 查看是否配置成功

hugo下载地址https://github.com/gohugoio/hugo/releases

二. 生成本地站点

  1. 输入命令

hugo new site quickstart

新建一个名为“quickstart”的网站

三. 添加主题

1. 输入以下命令下载默认主题

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

2. 将主题添加到站点配置

echo ‘theme = “ananke”’ » config.toml

四. 创建一篇文章

1. 输入以下命令，在 content/posts 目录下会自动生成 my-first-post.md 文件，此页面为文章初始页面。

hugo new post/my_first_blog.md（其中my_first_blog为文章的名字）

2. 打开 my_first_blog.md,默认内容为下：

title: "my_first_blog"
date: 2019-05-30T10:49:31+08:00
draft : true
以下是博客内容.....

其中draft 表示是否是草稿文件，编辑完成后请将其改为 false，否则编译会跳过草稿文件。

五. 预览文章的网页效果

1. 输入以下命令

hugo server -D

在代码底部位置会输出一个链接，点击此链接就可以浏览个人博客的网页效果

2. 配置config.toml

在文本编辑器中搜索 config.toml

出现以下页面

aseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

六. 生成静态页面并将其推送到GitHub

1. 输入命令

hugo -D

此时生成public目录

2. 在GitHub创建仓库

3. 将public目录push到GitHub创建的仓库中去

(1) 新建文件.gitignore

(2) 将public放进.gitignore

(3) 在public目录中执行git init使public自成仓库

(4) 在public目录git add .

(5) 在public目录git commit -v

(6) 最后git push将public目录推送到GitHub仓库
