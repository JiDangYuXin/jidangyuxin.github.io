如何用hugo搭建个人博客

如何用hugo搭建个人博客 本地安装git windows系统 一. 安装hugo 选择自己电脑对应的版本进行下载 下载安装完成之后配置好环境变量 到终端输入 hugo version 查看是否配置成功 hugo下载地址https://github.com/gohugoio/hugo/releases 二. 生成本地站点 输入命令 hugo new site quickstart 新建一个名为“quickstart”的网站 三. 添加主题 1. 输入以下命令下载默认主题 git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 2. 将主题添加到站点配置 echo ‘theme = “ananke”’ » config.toml 四. 创建一篇文章 1. 输入以下命令，在 content/posts 目录下会自动生成 my-first-post.md 文件，此页面为文章初始页面。 hugo new post/my_first_blog.md（其中my_first_blog为文章的名字）
